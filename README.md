# NEET Quizer Bot

Telegram bot for creating and taking NEET-style quizzes. Supports:
- Create quiz with title & description
- Add questions via Telegram Quiz polls
- Start quiz in DM or group
- Basic scoring (more features coming)

## Setup
1. Clone repo: `git clone https://github.com/God-zx/Neet-quizer.git`
2. Install deps: `pip install -r requirements.txt`
3. Create `.env` file with `BOT_TOKEN=your_token_here`
4. Run: `python bot.py`
